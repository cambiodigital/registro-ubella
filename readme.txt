== Description ==
Registro ágil de usuarios con gestión individual de perfiles. Cada usuario puede acceder, actualizar y mantener su información personal de manera independiente.

== Installation ==
1. Subir el plugin a la carpeta `/wp-content/plugins/`.
2. Activar el plugin desde el menú "Plugins" de WordPress.
